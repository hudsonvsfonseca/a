## 🚀 Como Executar o Projeto

### Pré-requisitos

- Node.js (versão LTS)
- Angular CLI

### Passos

bash
# Clonar o repositório
git clone https://github.com/seu-usuario/nome-do-repositorio.git

# Acessar a pasta do projeto
cd nome-do-repositorio

# Instalar as dependências
npm install

# Executar a aplicação
ng serve
